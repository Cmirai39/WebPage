Introduction about Cmirai

C言語・C++・JavaScript・HTML・CSS・JAVA・R言語・Pythonを学習中の初心者です.Pythonを主力としての機械学習を傍らにモバイルアプリ開発も視野にいれています.当サイトは紹介を兼ねた、HTML・JavaScript・CSSの練習です.

当サイトのコーディングについてよりよい方法がありましたら,Githubのissueでご指摘していただければ幸いです.当サイト関連ファイルはWebPageリポジトリのgh-pageブランチにあります.

ある程度使える言語

  • JAVA
  • JavaScript
  • HTML
  • CSS

Pythonに関して

Pythonは主に基礎構文,モジュールに関する学習をしています.また,機械学習で有名なTensorflowやKeras,Numpyを用いたディープラーニングの実装なども行っています.
Pythonによる主なプロダクトとしてクロスワードの問題を自動で生成するプログラムやイラストロジックを自動解析するプログラム、LINEbotやサーバー、APIを用いた自動取引のプラットフォームなどがあります.

JAVAに関して

JAVAでは主に基礎構文,Android開発に関する学習をしています.
JAVAの主なプロダクトはPythonで作ったイラストロジック解析プログラムの上位互換があります。実行にかかる時間は約30分の1程度に減少しています。

Web系に関して

主に基礎構文の学習を行っています。 主なプロダクトとしては当サイトがあります。
